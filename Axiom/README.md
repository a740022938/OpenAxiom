# Axiom

Axiom is a local AI data workspace.

Current phase: Phase 2 minimal data chain.

## Scope

This version only targets:

- Open project root
- Detect YOLO dataset
- Load images
- Load labels
- Draw bounding boxes
- Show annotation table

It does not implement training, AIP integration, OpenClaw integration, or complex AI features.

## Run

```powershell
cd E:\Axiom
py -3.11 -m venv .venv
.\.venv\Scripts\activate
python -m pip install -r requirements.txt
python main.py
```
